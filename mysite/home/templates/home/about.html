About Me

Abby Lacy:

Hello, I'm a senior computer science and women, gender and sexuality studies double major at DePauw University. I'm from northern Indiana, and my hometown is South Bend. I'm currenty building this site as a proof on concept when it comes to web development. Previously throughout my junior year of 2017-2018, I've been the resident intern at the Women's center on campus. I'm an huge fan of video games and spend most of my free time either coding or playing games. This coming semester I'm taking a web development and security course which I hope will further add to my skills to continue developing this site. ...

About this project:

This project is using Anaconda 3 and the Django web develeopment tools and kit using Python as a way to run most of the backend for this site. I've been using Atom as a development envionment, which has been useful tool for this project. I've been learning these tools on my own, and its been a long process of learning and trail to get this site up and running.
